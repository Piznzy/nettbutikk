# # MiniShop – Enkel Nettbutikk

## Beskrivelse
MiniShop er en enkel nettbutikk der brukere kan:
- Registrere konto og logge inn
- Se produkter
- Legge produkter i handlekurv
- Se ordre og totalpris

Prosjektet viser ferdigheter innen **utvikling, drift og brukerstøtte**, og er laget for å kunne demonstreres på eksamen.

---

## Teknologier
- **Backend:** Flask (Python)
- **Frontend:** HTML og CSS
- **Database:** MariaDB
- **Server:** Raspberry Pi eller Ubuntu VM

---

## Funksjonalitet
1. **Registrering/Login**
   - Brukere kan lage konto og logge inn for å bruke nettbutikken
2. **Produktvisning**
   - Liste over tilgjengelige produkter
3. **Handlekurv**
   - Legg produkter i handlekurv og se totalpris
4. **Ordrebehandling**
   - Handlekurv lagres som ordre for brukeren

---

## Brukerveiledning
1. Registrer deg som ny bruker eller logg inn  
2. Se tilgjengelige produkter på hjemmesiden  
3. Legg produkter i handlekurv  
4. Gå til “Handlekurv” for å se totalpris  
5. Trykk “Kjøp” for å fullføre ordren
